## 📋 Project Overview

The **Coral Heights Vehicle Registration Portal** is a fully client-side, multilingual HTML form designed for residents of **A Wing** to register their vehicles (cars, two-wheelers, bicycles) digitally and submit the data directly to the society committee via WhatsApp — eliminating paperwork delays.

---

## ✨ Features

- 🔐 **Secure login** — committee-controlled access with password protection
- 🌐 **Trilingual interface** — English · Gujarati · Hindi
- 🚗 **Multi-vehicle support** — Car, 2-Wheeler, and Bicycle registration
- 📱 **WhatsApp submission** — One-tap send to society secretary
- ✅ **Form validation** — Real-time required field checks
- 🎨 **Premium UI** — Navy + gold professional design
- 📲 **Mobile-first** — Fully responsive for all screen sizes
- ⚡ **Zero backend** — Works offline, no server required

## 🛠️ How to Use

1. Open `Coral-Heights-A-wing-late-submission-April-2026.html` in any browser
2. Login with committee-provided credentials
3. Fill in flat details and vehicle information
4. Review the summary
5. Tap **"Confirm & Send via WhatsApp"** — message auto-formats and opens WhatsApp

## 📋 Project Overview

This is a **fully digital, mobile-first vehicle registration form** built for **Coral Heights Housing Society, A Wing**. Residents can fill out their vehicle details digitally and submit instantly via **WhatsApp** — no paperwork, no delays.

Two versions are included:
- `coral_heights_vehicle_form.html` — **2025 Version** (English)
- `Coral-Heights-A-wing-late-submission-April-2026.html` — **2026 Version** (Trilingual: English + Gujarati + Hindi)

---

## ✨ Features

- 📱 **Mobile-First Design** — Works flawlessly on all screen sizes
- 💬 **WhatsApp Submission** — One-tap form send via WhatsApp API
- 🚗 **Multi-Vehicle Support** — Car, Two-Wheeler, Bicycle
- 🔢 **Quantity Spinner** — Register multiple vehicles per category
- 📝 **Review Modal** — Residents preview their form before sending
- ✅ **Real-time Validation** — Required fields checked before submission
- 🌐 **Trilingual Labels** — English, Gujarati (ગુજરાતી), Hindi (हिन्दी)
- 🏛️ **Committee Directory** — Direct WhatsApp & call buttons for society members
- 🎨 **Premium UI** — Navy, Coral & Gold theme with smooth animations
- 🖨️ **Digital-First Policy** — WhatsApp submission accepted; physical copy optional

## 🚀 How to Use

1. **Open** the HTML file on any mobile or desktop browser
2. **Fill** in your flat number, name, and contact
3. **Select** your vehicles (Car / 2-Wheeler / Bicycle)
4. **Tap** "Review & Send on WhatsApp"
5. **Preview** your form details in the modal
6. **Confirm** — WhatsApp opens with pre-filled message ready to send
